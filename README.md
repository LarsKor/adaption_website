# Adaption Website

Dit is een moderne, responsive Next.js + Tailwind CSS starter voor de Adaption-website. De content is origineel en in het Nederlands; het ontwerp is geïnspireerd door een professionele IT-consultancy site.

Belangrijk:
- Het contactformulier gebruikt Formspree (configureer je form ID in `components/ContactForm.js`).
- Deployment: Vercel wordt aangeraden (één klik deploy vanuit GitHub).

Scripts:
- `npm run dev` — start development server
- `npm run build` — bouw voor productie
- `npm start` — start productie server
